How to animate

A starters guide to animation, using simple examples!

With uniUI, animation is simple and quick, just like this:

Luau
uniUI.animate.custom(object, {BackgroundTransparency=1})

This specific function sets the object to a custom property set, overriding a currently set "state", the function accepts the object and a properties list.
