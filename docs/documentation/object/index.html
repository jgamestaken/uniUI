uniUI.object

This submodule is responsible for the creation of UI objects such as Frame and TextLabel, among many more!

new()

Luau
uniUI.object.new(type, base, properties, untweenable_properties)

This function is used to generate new UI objects.

Property

Expected Value

Example

type

String with object type

"Frame"

base

Object that will be used as Parent

-

properties

Table of properties

{BackgroundTransparency=1}

untweenable_properties

Table of untweenable properties

{ZIndex=1}

Luau
uniUI.object.import(path, base)

This function is used to clone a UI module from a path and set it up to work with uniUI.

Property

Expected Value

Example

path

toggle_activation()

Just about works

This function just about works, and it's up for improvement in later versions in uniUI.

Luau
uniUI.object.toggle_activation(object, toggle)

Toggles the activation of an object, activation determines whether or not Tweens can be executed on that object. Useful for when using a closing animation.

Property

Expected Value

Example

object

Object(+ descendants) that will be toggled

-

toggle

Bool to set activation to

true
